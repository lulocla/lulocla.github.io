About this Subject

Statistics allows us to build models for situations where relationships between variables are not exact but approximate — I = f(X) + E, where E represents random error. This module focuses on linear models, studying the relationship between a response variable and continuous (regression), categorical (ANOVA), or mixed (ANCOVA) explanatory variables for both explanatory and predictive purposes.

Contents
  • Introduction — linear model, Gauss-Markov conditions
  • Estimation — matrix representation, least squares, QR decomposition, estimable parametric functions
  • Inference — model contrasts, hypothesis testing, confidence intervals, permutation tests
  • Regression — simple and multiple linear regression, goodness-of-fit, predictions, autoregression
  • Diagnostics — residual types, homoscedasticity, normality, outliers, leverage, multicollinearity
  • Alternative methods — transformations, polynomials, robust regression, logistic regression
  • Variable selection and regularisation — stepwise, Ridge, LASSO, PLS, principal component regression
  • Categorical predictors — ANOVA, ANCOVA, factor coding
Skills Gained
  • Linear model fitting and interpretation
  • Simple and multiple regression in R
  • Model diagnostics — residuals, outliers, leverage
  • ANOVA and ANCOVA
  • Logistic regression
  • Ridge regression and LASSO regularisation
