Software for Data Analysis

Compulsory  |  5 ECTS  |  Code: M0.153  |  18 Feb 2026 – 1 Jul 2026

Professor: Joan M. Nuñez Do Rio

About this Subject

This module focuses on learning and using the statistical package R and related tools — including R-Commander, RStudio, and R Markdown. R is one of the most flexible and powerful tools for data analysis, freely available and maintained by the international scientific community. The module covers data management, statistical analysis, graphics, and programming in R, all within the context of bioinformatics and biostatistics.

Contents
  • Introduction to R, RStudio and R Markdown — installation, interface, package management, data import/export
  • Descriptive statistics and graphics — ggplot2, frequency tables, linear regression, ANOVA
  • Programming fundamentals — operators, data structures, functions, database access in R
  • Probability and simulation — probability distributions, sampling, simulation with R
  • Introduction to machine learning with R — supervised, unsupervised learning, deep learning, ANOVA
  • R packages for bioinformatics — Bioconductor, reproducible environments, Shiny
Skills Gained
  • Statistical computing with R (RStudio, R Markdown, R Commander)
  • Data visualisation with ggplot2
  • Programming in R — functions, data structures, control flow
  • Probability simulation and sampling
  • Introduction to machine learning with R
  • Bioconductor and bioinformatics R packages
