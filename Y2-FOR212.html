FOR212 Investigative Analytical Science

Description

This module is designed to teach basic analytical skills,the theory behind these techniques, and an understanding of their application in forensic and biomedical contexts

Aims

The aim of this Module is to provide the student with an understanding of basic analytical techniques.

Learning Outcomes

By the end of this module I was able to:

  1. Select, describe and apply analytical methods
  2. Apply, interpret and report statistical tests for data analysis
  3. Carry out a range of specified analyses
Project
Colorimetric assay of serum protein (Laboratory report)
Statistics exercises
Chemistry questions
Key Transferable Skills for this module are:

Communication, Team Work, Problem Solving, ICT Skills

Indicative content
  1. Quality

    2. Sample custody and traceability. Data interpretation and reporting.

  2. ⁠Sample Preparation

    3. Sampling. Drying. Crushing. Weighing. Dissolution. Separation and concentration.

  3. ⁠ ⁠Spectroscopy

    4. Electromagnetic radiation and the nature of light. Beer Lambert law. Theory of spectroscopy. Instrumentation. UV-visible, AAS spectroscopy. Presumptive tests. Spectroscopic techniques.

  4. Theory and Principles of Chromatography

    5. Introduction to TLC.

  5. Immunotechnology

    6. Diagnostic and detection systems based on antibody-antigen interaction: RIA, ELISA, Agglutination, Precipitation reactions. Direct, Indirect and sandwich assays. Use of serological methods in diagnosis of infectious diseases and forensic applications such as body fluid identification
