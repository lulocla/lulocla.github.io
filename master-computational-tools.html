Computational Tools for Bioinformatics

Optional  |  5 ECTS  |  Code: M0.151  |  18 Feb 2026 – 1 Jul 2026

Professor: Josep Jorba Esteve

About this Subject

Bioinformatics research typically involves large-scale genomic analysis of massive datasets — processes that cannot be carried out efficiently by hand. This module introduces the core technical infrastructure needed: GNU/Linux operating systems, shell scripting, workflow management, containerisation, and relational and non-relational database management. It is an essential introductory module from a technical standpoint for all subsequent bioinformatics modules.

Contents
  • GNU/Linux environment setup on a virtual machine — terminal, basic commands, file system management, bioinformatic data download and manipulation
  • Environment and container management — Docker and container workflows
  • Workflow management — Snakemake and Nextflow
  • Relational and non-relational database design — MySQL tables, keys, data management for biological services
Skills Gained
  • GNU/Linux command-line management
  • Shell scripting for bioinformatics data processing
  • Container management with Docker
  • Workflow automation (Snakemake, Nextflow)
  • Relational database design with MySQL
  • Non-relational database management
