Computational Genomics

Compulsory  |  5 ECTS  |  Code: M0.154  |  18 Feb 2026 – 1 Jul 2026

Professor: Eduard Ocaña Pallarès, Agnès Pérez Millan

About this Subject

This module provides an in-depth introduction to the analysis and description of the different elements encoded in a genome. Genomics has undergone a dramatic evolution over the past two decades. Students learn to access genomic information online, understand bioinformatic applications for genome annotation, and appreciate that genomic knowledge is continuously evolving — with gene catalogues, regulatory elements, and other genomic features being refined daily.

Contents
  • Genomic servers — UCSC and Ensembl genome browsers, navigation philosophy
  • Sequence alignment — alphabets, global/local alignment, scoring matrices, multiple alignments, BLAST, genome alignments
  • Genome annotation — genomic and epigenomic structure, gene anatomy, regulatory regions, gene expression regulation
  • Massive sequencing — transcriptome sequencing (RNA-seq), ChIP-seq, ATAC-seq, Hi-C
  • Single-cell sequencing — scRNA-seq, scATAC-seq
  • Genomic variant analysis — GATK, CRISPR/Cas9 screens, public annotation consortia data
Skills Gained
  • UCSC and Ensembl genome browser navigation
  • Sequence alignment algorithms (BLAST, dynamic programming)
  • Genome annotation and gene structure analysis
  • RNA-seq, ChIP-seq and ATAC-seq data interpretation
  • Genomic variant analysis with GATK
  • Use of public genomic databases and annotation consortia
