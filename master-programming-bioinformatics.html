Programming for Bioinformatics

Compulsory  |  5 ECTS  |  Code: M0.152  |  18 Feb 2026 – 1 Jul 2026

Professor: Pau Perea Paños

About this Subject

Bioinformatics was born over two decades ago from the need to analyse large volumes of biological data — particularly following the sequencing of the first complete genomes. This module develops algorithmic thinking and Python programming skills specifically applied to bioinformatics problems, using libraries such as Biopython for DNA/RNA sequence analysis, scientific computing, and software quality testing.

Contents
  • 1. Introduction to Python programming — syntax, first programs, data management
  • 2. Python fundamentals — control flow, functions, logical structures
  • 3. Scientific libraries in Python — mathematical and graphical libraries
  • 4. DNA, RNA, sequences and motifs — Biopython for biological sequence analysis
  • 5. Software testing and quality — unit testing, object-oriented programming applied to bioinformatics
Skills Gained
  • Python programming — syntax, control flow, functions
  • Scientific libraries (NumPy, Matplotlib)
  • Biopython for DNA/RNA sequence analysis
  • Bioinformatics file format handling
  • Software testing and quality assurance
  • Object-oriented programming for bioinformatics
